👻 LEGION GO — HACKING RIG

Step-by-step setup guide

← MOLLY//CORE
01
📦 Install Kali Linux
Open PowerShell as Administrator on the Legion Go:
PowerShell (Admin)
wsl --install -d kali-linux
Then: Restart when Windows asks. Open Kali from Start Menu. Set a username + password — remember it!
02
🔧 Install Hacking Tools
In the Kali terminal, paste this entire command:
Kali Terminal
sudo apt update && sudo apt install -y nmap nikto gobuster sqlmap hydra curl python3 git wget netcat-openbsd whois dnsutils
Takes 3-5 mins. Type password when asked — characters are hidden, that's normal.
Test it worked
nmap --version
Should say Nmap 7.xx — you're done ✅
03
🤖 Install Offline AI
Back in regular Windows PowerShell (not Kali):
Windows PowerShell
winget install Ollama.Ollama
Restart terminal after install, then pull the model:
ollama pull phi3:mini
2.3GB download. Takes 5-10 mins on first run.
Test it
ollama run phi3:mini
Type "hello" — if it responds you're done. Ctrl+D to exit.
04
🎯 Test Your Rig
In Kali, scan a legal test target:
Kali Terminal
nmap -sV scanme.nmap.org
If you see port results — your rig is ready! 🎉

🔥 Troubleshooting
winget not found
Windows Store → search "App Installer" → install → restart
wsl not found
Run Windows Update first, then try again
Kali won't open
Restart after install, look in Start Menu for "Kali Linux"
Password not working
Characters are invisible as you type — just type slowly and hit Enter
Ollama slow
Normal on first pull — it's downloading 2.3GB, just wait
